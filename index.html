Inicio

En esta actividad práctica, tendrás la oportunidad de poner en acción los conceptos aprendidos sobre HTML y CSS para desarrollar una página web básica. Utilizando las etiquetas y técnicas de formato vistas en la Unidad II, crearás una página que no solo sea funcional, sino también visualmente atractiva. Esta tarea te permitirá reforzar tus habilidades en diseño web, abordando la estructura y el estilo de una página web sencilla. Al finalizar la actividad, habrás creado una página que cumpla con los estándares básicos de desarrollo web, manejando correctamente el código HTML para la estructura y el CSS para mejorar su apariencia. El proyecto se evaluará tanto por su funcionalidad como por su diseño, tomando en cuenta el uso apropiado de etiquetas, la correcta disposición del contenido, y la coherencia estética lograda con CSS.
